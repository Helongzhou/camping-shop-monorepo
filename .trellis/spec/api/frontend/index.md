# Frontend Guidelines — Not Applicable

> `@myshop/api` is a backend-only NestJS package.

---

## Overview

There is no frontend code under `apps/api/`. All UI lives in `@myshop/web` (`apps/web/`).

Trellis generated `spec/api/frontend/` scaffolding for monorepo symmetry. **Do not load these files for API implementation tasks** unless building admin UI inside the API repo (not planned).

For frontend conventions, use `spec/web/frontend/` instead.
